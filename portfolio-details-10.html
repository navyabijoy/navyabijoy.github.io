Memoria - AI Flashcard Generator

As part of my Software Engineering Fellowship with Headstarter, I developed Memoria, an innovative web app designed to generate customized flashcards based on any topic you enter. This project aims to make studying more efficient and effective, assisting users in exam preparation, language learning, or exploring new subjects. Memoria is built using a modern tech stack to ensure a seamless user experience.

Features:
  • Interactive Interface: Designed a user-friendly web application that allows users to input topics and receive personalized flashcards.
  • Efficient Study Aid: Generates flashcards tailored to the user's input, helping streamline learning and retention.

Usage:

  1. Open your browser and visit the Memoria web app.
  2. Enter the desired topic for which you want to create flashcards.
  3. Review and use the generated flashcards to aid in your study sessions.

Technologies Used:

  • React & Next.js: For building the responsive and dynamic web application.
  • Tailwind CSS: For designing a sleek and modern user interface.
  • Clerk: Integrated for seamless user authentication and management.
  • Open Router: Used for enhanced API interactions and responses.
  • Firebase: Backend database for storing user data and ensuring quick response times.
